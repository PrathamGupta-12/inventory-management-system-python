# Inventory Management System

A simple console-based Inventory Management System built using Python.

## Features

- Add a product
- Update product quantity
- Remove a product
- Search for a product
- Display all inventory items
- Calculate total inventory value

## Technologies Used

- Python 3
- Dictionaries
- Lists
- Object-Oriented Programming (Basic)

## How to Run

```bash
python main.py
```

## Note

This is one of my early Python projects. The main goal was to practice Python, dictionaries, and object-oriented programming concepts. I plan to improve the code structure and add more features as I continue learning.
